# What

Repo that holds various files created while reading Kneusel's book "Strange Code."

# How

To compile and run, look for the appropriate langauge.

## Forth

Files ending in "*.f"

```
sudo get-apt install gforth

gforth fizzbuzz.f

run

bye
```

## Snobol

Files ending in "*.sno"

Donwload snobol4 and go to directory.

```
./snobol4 hello.sno
```
